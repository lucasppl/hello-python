# Hello, Python! 🐍

**Aprenda Python do zero ao primeiro projeto!**  
> Um projeto de extensão gratuito feito por alunos da **Universidade Católica de Brasília** para os alunos dos **Centros de Línguas (CILs)** de Brasília.

![Demo](assets/demo.gif)

O **Hello, Python!** não é apenas um curso de programação; é uma ponte entre a tecnologia e o aprendizado de línguas. Desenvolvido por estudantes de **Análise e Desenvolvimento de Sistemas (ADS)**, nosso foco é ensinar lógica de programação e Python de forma unificada com o inglês técnico, preparando jovens para o mercado global.

---

## 🚀 O que é o projeto?

**Acreditamos que aprender a programar é como aprender um novo idioma.** Como a sintaxe do Python utiliza termos em inglês (como `if`, `else`, `while`), transformamos nossas oficinas em um laboratório prático onde o aluno exercita a lógica e expande seu vocabulário simultaneamente.

### Por que participar?
- **100% Gratuito:** Feito de estudantes para estudantes. Da comunidade para a comunidade.
- **Do Zero:** Não precisa de experiência prévia com código.
- **Certificado:** Receba reconhecimento pela sua evolução e projeto final.

---

## 🛠️ Projetos Reais

Ao final das oficinas, cada aluno desenvolve seu próprio miniprojeto para colocar no portfólio:
- 🎮 **Snake Game:** Criando jogos com Pygame.
- 📝 **Quiz de Inglês:** Testando conhecimentos com listas e dicionários.
- 🧮 **Calculadora Inteligente:** Aplicando funções e lógica matemática.
- 🤖 **Automações Simples:** Facilitando tarefas do dia a dia.
