O que é o Hello Python?

Um projeto de extensão gratuito que ensina Python para alunos dos Centros de Línguas de Brasília, combinando o inglês da escola com programação — as duas habilidades mais valorizadas no mercado.

Preciso ter experiência em programação?

Não! Começamos do zero. Do que é uma variável até criar seu primeiro jogo. Basta curiosidade e vontade de aprender.

Preciso de computador para participar?

As aulas têm computadores disponíveis. Para estudar em casa, use o Replit.com — funciona no navegador, até no celular.

As aulas são presenciais ou online?

Formato híbrido: aulas presenciais no CIL + materiais e exercícios online para praticar quando quiser.

Como funciona o certificado?

Conclua todos os módulos, entregue seu projeto final e receba seu certificado de conclusão.

É realmente gratuito?

100% gratuito. O Hello Python é um projeto de extensão universitária para democratizar tecnologia. Nenhum custo para alunos dos CILs.
