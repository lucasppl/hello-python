Gmail

hellopython.ucb@gmail.com
  1. Compacte seu projeto em um arquivo .zip
  2. (Botão direito na pasta ➞ Compactar para ➞ Arquivo ZIP)
  3. Abra seu aplicativo de e-mail
  4. Anexe o arquivo .zip
  5. Informe seu nome completo + projeto escolhido
  6. Envie para o e-mail oficial do curso
Abrir Gmail

Certifique-se de anexar o arquivo .zip antes de enviar.

Mais fácil de organizar e corrigir.

Envio pelo

WhatsApp

Caso tenha dificuldades para enviar por e-mail, você pode enviar o arquivo .zip pelo WhatsApp.

  1. Compacte seu projeto em um arquivo .zip
  2. (Botão direito na pasta ➞ Compactar para ➞ Arquivo ZIP)
  3. Abra a conversa do WhatsApp
  4. Envie o arquivo no nosso grupo do WhatsApp
  5. Informe seu nome completo
  6. Informe o projeto escolhido
Abrir WhatsApp

Envie também seu nome completo e o nome do projeto escolhido.

Utilize esta opção apenas caso tenha dificuldades com o envio por e-mail.
